Project directory for mini data analysis project HW1 for BIOS 731

Imaging data is from Robin Carhart-Harris et al and was obtained from OpenNeuro (https://openneuro.org/datasets/ds003059/versions/1.0.0). Images were already pre-processed before download.

Brain volumes 50 through 100 were extracted using script Shrink Image.R found in the data folder to reduce file size.
